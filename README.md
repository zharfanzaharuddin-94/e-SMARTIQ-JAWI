**e-SMARTIQ JAWI** ialah platform pembelajaran Jawi berasaskan web yang dibangunkan bagi membantu guru dan murid sekolah rendah meningkatkan kemahiran membaca Jawi melalui latihan interaktif, analisis prestasi dan penilaian digital.

## Objektif

* Membantu murid meningkatkan kemahiran membaca Jawi.
* Memudahkan guru memantau perkembangan murid secara digital.
* Menyediakan analisis prestasi yang mudah difahami.
* Mengurangkan penggunaan rekod penilaian secara manual.

## Ciri-ciri Utama

* Portal Guru
* Portal Murid
* Latihan Bacaan Jawi
* Penilaian Sebutan
* Dashboard Analisis Keseluruhan
* Profil Prestasi Murid
* Graf Trend Kemajuan
* Kad Laporan Digital Murid
* Perpustakaan Jawi
* Pengurusan Data Murid
* Eksport Laporan PDF

## Teknologi Digunakan

* HTML5
* CSS3
* JavaScript (Vanilla)
* IndexedDB
* Google Apps Script
* Google Sheets
* Google Cloud Text-to-Speech
* Gemini AI (Analisis Kandungan)

## Struktur Sistem

* Portal Guru
* Portal Murid
* Perpustakaan Jawi
* Analisis Keseluruhan
* Profil Prestasi Murid
* Modul Latihan e-SMARTIQ JAWI

## Kegunaan

Sistem ini sesuai digunakan oleh:

* Guru Pendidikan Islam
* Murid Sekolah Rendah
* Sekolah yang melaksanakan pengajaran dan pentaksiran Jawi secara digital
